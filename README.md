# Sistema Web para Restaurante

## Descripción
Sistema web bilingüe (español/ruso) para restaurante con:
- Catálogo de platos
- Personalización de platos
- Sistema de pedidos sin pago online
- Panel proveedor con notificaciones
- Analíticas y reseñas

## Stack Tecnológico
- **Backend:** Node.js, Express, PostgreSQL, Socket.io
- **Frontend:** Next.js 14, React, Tailwind CSS
- **DevOps:** Docker, GitHub Actions, Nginx

## Estructura del Proyecto
- `/backend` - API y lógica de negocio
- `/frontend` - Aplicación web cliente
- `/docs` - Documentación
- `/deployments` - Configuración deploy
